參與本班學員，符合『勞動部青年就業讚計畫』者(課程代碼: E23432)，學費可獲政府100%補助，完全免費。 最近一期結訓班級，就業率高達90%，青讚學員100%就業。 恭喜呂同學，順利進入美商擔任軟體工程師，薪資達55,000元。 《遠見雜誌》「2016上市櫃公司最愛用大學生調查」，台北科技大學位居第三，人才培育深獲業界肯定。 前期招生滿班，本期名額有限，敬請踴躍報告 。

Software Engineer

軟體設計是當前最具前(錢)途的職業 據US News研究報告指出2014年美國前百大最佳工作中，軟體工程師(software developer）、電腦系統分析師(Computer Systems Analyst)與網站開發者(Web Developer)分列第一名、第二名與第九名。另根據美國勞工部門調查，軟體工程師平均年薪 90,060 美金(約280萬台幣)，最頂尖的 10% 甚至可達 13.8 萬美金(約427萬台幣)。可見程式設計師在經濟不景氣的年代，已經逆向成為最有前（錢）途的職業之一。 軟體開發人才供不應求 隨著巨量資料、雲端應用服務、行動創意App、電子商務、物聯網、網路遊戲、數位娛樂、RTB 廣告、穿戴式裝置、與電動車等新興產業的蓬勃發展，造成軟體人才的需求還會不斷提昇，廠商大嘆人才難求。 Java程式已成軟體開發的主要工具 近年來Java已經是許多大型系統升級轉換的第一考量語言，從金融、財稅、醫療、電信、戶役政等電腦系統都已經改用Java開發，連各式各樣的雲端系統應用都是以Java為主要工具，現在的程式設計已經不能不懂Java。 行動App產業將帶動另一波經濟榮景 隨著智慧行動設備普及化，行動應用服務需求緊迫，App產業儼然已成下一波經濟發展明星。根據國外知名網站Flurry調查顯示，2010年6月，從2008年蘋果電腦推出線上市集起開始計算，行動App產業只花3年，人們每日使用App平均時間已超過瀏覽網頁平均時間，徹底改變了人類建立超過10年的使用習慣。App經濟將為台灣科技業帶來了機會，它的崛起也將讓App程式開發人才成為職場寵兒。 臺北科大推廣中心精心設計就業養成課程 為充分滿足雲端服務、巨量資料與智慧行動應用等新興產業及各行各業科技化服務所需之優質軟體技術人力，特別依據產業職能需求規劃Java & Android程式設計師養成班，讓您完成培訓之後，具備資訊系統規劃與設計、Java程式撰寫、Android App 開發、企業資料庫設計與IT專案管理管理等能力，受訓期滿將提供就業輔導，協助學員找到理想的工作。

Teacher

課程負責人曾任職資策會中壢中心(現改名國際人才發展中心)主任，具15年以上之就業養成訓練與輔導經驗，曾締造就業百分百與過半進上市公司之結訓班級紀錄。 ․培訓內容涵蓋Java程式設計、Android App軟體開發、雲端網路與專案管理技術領域之就業養成班，與產業發展趨勢同步。 ․課程設計內容除含趨勢產業所需之關鍵核心技術外，亦規劃以培養實作、解決問題、協同合作、學習、溝通與表達能力等軟性能力的精緻實務教案和專題，全方位滿足企業求才的必要需求。 ․資深業界名師親自授課，帶領進入未來10年最夯工作職位。 ․提供個別化之學習輔導與職涯顧問諮詢；以人本關懷為重，對學員進行適性適才的學習服務、生涯規劃及就業輔導，使學員建立信心並發揮個人優勢，輔以累積多年來的業界人脈，對結訓學員進行就業輔導 （履歷撰寫、面試技巧…等指導），提升職場競爭力。 ․輔導學員取得Oracle OCPJP、CompTIA Network+與CompTIA Project+等國際專業認證。

參加本班的訓練，需要具備哪些背景與條件？ 答：無需任何資訊背景，只要您是大專畢業，不限科系，對從事Java程式設計與智慧行動App開發工作有興趣者皆可報名參訓本班。
本班結訓後之就業領域與工作內容為何？ 答：本課程之主要訓練目標在於培養雲端服務、巨量資料、智慧行動應用等新興產業與各行各業科技化服務所需之Java軟體工程師;使學員具備系統規劃、Java程式撰寫、Android App 開發、可從事的工作內容包括： Java 程式設計、Android App程式設計、企業資料庫設計與管理、資訊系統分析與設計、與資通訊產品之業務和行銷等項目。
從本班結訓時，是否會有就業輔導的支援服務？ 答：本中心在您結訓前的一個半月，開始提供相關的就業資訊與個別化之職涯顧問諮詢，俾利學員順利找到理想的工作。由於本校百年來持續不斷培育優質人才，協助產業創造經濟奇蹟，已經累積優異的訓練成效與口碑，並與許多廠商有過良好的求才配合經驗，所以知名企業或廠商均競相錄用本校所培育的學生，這對於原本非相關科系背景的資訊新鮮人而言，更是轉換職場跑道的轉捩點與絕佳良機。
企業求才時，對求職者之主要評估要素為何? 答：主要評估要素包含下列三項: 1. 專業基本素養:工作領域之基本專業知識、技術、整合與實作能力 2. 軟性技能: 學習能力、解決問題能力、溝通技巧、創新與協同合作 3. 優質工作態度:積極主動、肯負責任與高配合度…
